# RoboSec: ROS Cybersecurity
Cybersecurity research on ROS (Robot Operating System)

## Current Project 
 SoK (Systemization of Knowledge) paper on ROS Cybersecurity

## Dataset Info

**Search String** 

("ROS" OR "Robot Operating System") AND ("cybersecurity" OR "attack" OR "threat" OR "vulnerability")

**Inclusion Criteria**

Year: 2016-2026
Database: 	IEEE Xplore, ACM Digital Library, Springer Nature
USENIX Security, ACM CCS, IEEE S&P, NDSS 
Publication Type: Conferences, Journals 
Peer-reviewed venues

**Exclusion Criteria**

IEEE Access, MDPI, Not ROS, Not Cybersecurity (Security Intelligence/Surveillance Applications, Military security, Human / Physical security/safety/reliability, Privacy)


**#(ROS + Cybersecurity) papers (Title + Abstract screening):**

**ACM** : 11/862,     **IEEE** : 71/366,     **Springer** : 16 /1000

**Current analysis results:**

Categorized papers based on S (Spoofing), T (Tampering), R (Repudiation), I (Information Disclosure), D (Denial of Service), E (Elevation of Privilege), IsThreat (presents attacks, discusses vulnerabilities), IsMitigation (proposes detection, mitigation, security features)
